# Contributing
The `src/index.ts` file is generated by `prebuild/index.gen.mjs` and should therefore not be modified manually.
You can trigger the generation process by running `npm run prebuild`.

To add additional error messages, simply add them to the `src/data/errors.json` file.
The structure is defined in the file itself.

## Interfaces and types
If the `CustomBaseErrorOptions` and `CustomAdditionalErrorDetails` types are extended with additional properties, 
the `README.md` file should be updated in the _**Interfaces and types**_ section to include those properties.
